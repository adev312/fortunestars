# Fortune Stars School Portal

A responsive school portal UI concept built with HTML, CSS, and JavaScript. It showcases a modern academic dashboard experience with role-based views for students, parents, teachers, and administrators.

## Features
- Clean login screen with role selection
- Student dashboard with results, schedule, and assignments
- Parent, teacher, and admin viewing experience
- Responsive layout for desktop and mobile screens
- Interactive UI elements for a polished demo experience

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Goals
This project was created as a frontend portfolio demo to highlight:
- UI/UX design thinking
- Responsive web layout
- Interactive component behavior
- Clean, modern styling

## How to Run
1. Open the project folder in your browser, or
2. Use a simple local server such as VS Code Live Server

## Publish on GitHub Pages
1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Under Source, select Deploy from a branch.
5. Choose the main branch and the root folder.
6. Save and wait for GitHub to publish the site.

## Notes
This project is a frontend demo and does not connect to a real school database or authentication system.
